# typecho-studio
# 🚀 Typecho Content Studio 使用手册

本软件是一款专为 Typecho 打造的集成化智能创作工作站，集成了 AI 润色、多媒体上传、全站备份等核心功能。

## 🛠 快速上手
1. **连接服务器**：
   - 在右侧“连接配置”中输入：`博客地址`（如 blog.com）、`账号`、`密码`。
   - 点击【同步配置并连接】，下方日志显示“同步成功”即表示连接就绪。
2. **撰写与发布**：
   - 在编辑器输入标题和内容（支持 Markdown）。
   - 选择右侧的“分类”，设置“状态”。
   - 点击底部的【确认提交文章】。
3. **本地同步**：
   - 双击“本地仓库”中的文件可加载草稿。
   - 双击“远程管理”中的文章可直接从服务器拉取并编辑。

## 📂 多媒体上传说明
- **操作**：点击【上传图片附件】即可选择文件。
- **智能排版**：
  - **图片**：上传后自动居中。
  - **音视频**：插入后可在网页端直接通过 HTML5 播放器播放。
  - **档案**：自动生成带图标的下载引用块。

## 🤖 AI 智能助手
- **配置**：在连接配置处填入 DeepSeek API Key。
- **功能**：点击【AI 智能润色正文】，系统将自动优化段落表达、修正错别字并美化排版。
- **注意**：DeepSeek-Reasoner 模型思考时间较长，请耐心等待日志反馈。

## 🛡 数据安全
- **全站备份**：建议每周点击一次【一键全站本地备份】，文章将以 Markdown 格式保存在 `backups` 文件夹中。
- **自动保存**：软件每 60 秒会自动保存当前草稿至 `content/drafts`。


# 配置指南
# ⚙️ API 与 服务器配置指南

### 1. Typecho 服务器配置
- **XMLRPC 开启**：确保你的 Typecho 并非极度阉割版，默认情况下 Typecho 是开启 XMLRPC 的。
- **上传限制**：
  - 若上传视频/大文件失败，请修改服务器 `php.ini` 中的 `upload_max_filesize` 和 `post_max_size`。
  - 在 Typecho 后台 -> 设置 -> 基本 -> 允许上传的文件类型中，手动补全：`webp, mp4, zip, pdf, docx, xlsx` 等。

### 2. DeepSeek AI 配置
- **获取 Key**：访问 [DeepSeek 开放平台](https://platform.deepseek.com/) 申请。
- **API 地址**：软件默认已配置为 `https://api.deepseek.com/chat/completions`。
- **余额提醒**：如果 AI 报错 `402` 或 `insufficient balance`，请检查账户余额。

### 3. 本地目录说明
- `/content/drafts`: 存放自动保存和手动保存的草稿。
- `/content/sent`: 存放成功发布的文章记录。
- `/backups`: 存放全站备份产生的 Markdown 集合。
- `config.yaml`: 存储你的连接信息（加密存储密码，但建议不要在公共电脑上操作）。
- `studio_log.txt`: 记录所有操作历史，排查错误时请查阅此文件。
